# 🎬 BollywoodGPT

### AI-Powered Bollywood Assistant using Groq, RAG, FAISS & Streamlit

An intelligent Bollywood chatbot that answers questions about **movies, actors, songs, and the latest Bollywood news** using **Retrieval-Augmented Generation (RAG)**, **Groq Llama 3.3**, **FAISS**, **Wikipedia**, and **HuggingFace Embeddings**.
